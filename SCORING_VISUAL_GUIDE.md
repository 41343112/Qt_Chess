# 吃子分數功能視覺指南 (Scoring Feature Visual Guide)

## UI 位置 (UI Location)

吃子分數顯示位於遊戲視窗的右側面板，在時間控制設定上方。

The score display is located in the right panel of the game window, above the time control settings.

│ 棋譜列表           │  棋盤區域  │  吃子分數                         │
│                    │            │  ┌────────────────┐               │
│ 1. e4 e5          │     □■□■□■□■│  │ 白方: 0        │               │

## 分數更新範例 (Score Update Example)

### 範例 1：白方吃掉黑方的騎士
**Example 1: White captures Black's knight**

```
移動前 (Before move):
┌────────────────┐
│ 白方: 0        │
└────────────────┘
┌────────────────┐
│ 黑方: 0        │
└────────────────┘

移動後 (After move):
┌────────────────┐
│ 白方: 3        │  ← 增加 3 分（騎士價值）
└────────────────┘
┌────────────────┐
│ 黑方: 0        │
└────────────────┘
```

### 範例 2：黑方吃掉白方的皇后
**Example 2: Black captures White's queen**

```
移動前 (Before move):
┌────────────────┐
│ 白方: 3        │
└────────────────┘
┌────────────────┐
│ 黑方: 5        │
└────────────────┘

移動後 (After move):
┌────────────────┐
│ 白方: 3        │
└────────────────┘
┌────────────────┐
│ 黑方: 14       │  ← 增加 9 分（皇后價值）
└────────────────┘
```

### 範例 3：多次吃子後的累計分數
**Example 3: Accumulated scores after multiple captures**

```
遊戲進行中 (During game):
┌────────────────┐

```
┌──────────────────────────────────────┐
│ 棋子類型          │ 價值              │
├──────────────────────────────────────┤

```
白方兵通過吃過路兵吃掉黑方兵：
Before:    After:
□ ■ □      □ ■ □
♟ ♙ □  →  □ ♙ □   白方分數 +1

```

### 兵升變後被吃 (Promoted Pawn Captured)
```
白方兵升變為皇后後被黑方吃掉：
┌────────────────┐
│ 黑方: 9        │  ← 按照皇后的價值計分，而不是兵
└────────────────┘
```

## UI 樣式說明 (UI Style Description)

- **字體大小**：12pt，粗體
- **對齊方式**：置中
- **背景顏色**：淺灰色 (#f0f0f0)
- **內邊距**：5px
- **圓角**：3px
- **即時更新**：每次吃子後立即更新，無需手動刷新

**Font size**: 12pt, bold
**Alignment**: Center
**Background color**: Light gray (#f0f0f0)
**Padding**: 5px
**Border radius**: 3px
**Real-time update**: Updates immediately after each capture, no manual refresh needed

## 使用建議 (Usage Tips)

1. **策略參考**：使用分數來評估當前局勢的優劣
2. **教學工具**：幫助初學者理解棋子的相對價值
3. **比賽記錄**：在對局結束後可以看到雙方的吃子統計

**Strategy reference**: Use scores to evaluate the current position advantage
**Teaching tool**: Helps beginners understand relative piece values
**Match record**: View capture statistics at the end of the game
