## Features Demonstrated

### 1. Time Control Panel (Right Side)
- **Group Box Title**: "時間控制 (Time Control)"
- **Total Time Dropdown**: Combo box with options from unlimited to 60 minutes
- **Increment Slider**: Horizontal slider (0-60 seconds) with current value display
- **Timer Displays**: Large, prominent time displays for both players

### 2. Timer Display States

**Inactive (No Time Control)**
```
┌──────────────────────┐
│      --:--           │
└──────────────────────┘
```

**Active Player Timer (Green Background)**
```
┌──────────────────────┐
│ █    04:37   █       │  ← Green background (#4CAF50)
└──────────────────────┘
```

**Inactive Player Timer (Gray Background)**
```
┌──────────────────────┐
│      05:03           │  ← Gray background (#333)
└──────────────────────┘
```

## Responsive Design

The layout uses Qt's layout managers:
- **Horizontal Layout**: Board on left (stretch factor 1), controls on right (stretch factor 0)
- **Board Layout**: Maintains square aspect ratio and scales with window
- **Right Panel**: Fixed width, allowing board to use remaining space
- **Minimum Window Size**: 520x380 pixels (accommodates time controls)
- **Default Window Size**: 900x660 pixels (optimal viewing)

## Color Scheme

- **Active Timer Background**: #4CAF50 (Green) - High contrast, easy to identify
- **Inactive Timer Background**: #333 (Dark Gray) - Subtle, non-distracting
- **Timer Text**: #FFF (White) - High contrast for readability
- **Timer Border**: Rounded corners (5px) for modern appearance
- **Font**: Bold, 16pt for timer values
