TECHNICAL IMPLEMENTATION
================================================================================

AspectRatioWidget:
- Custom Qt widget maintaining 1:1 aspect ratio for child widget
- Uses hasHeightForWidth() and heightForWidth() (Qt's standard mechanism)
- Manual geometry positioning in resizeEvent() for precise centering
- Reusable component for future features

Layout Changes:
- Changed from QVBoxLayout (vertical) to QHBoxLayout (horizontal)
- Left panel uses QSizePolicy::Preferred with min/max width constraints
- Board container uses AspectRatioWidget with stretch factor 1
- Simplified updateSquareSizes() logic
