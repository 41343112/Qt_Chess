## 顏色方案

### 背景
- 深藍黑色 (#0A1628)
- 營造深邃的科技感

### 標題
- 文字顏色：霓虹青色 (#00FFFF)
- 背景：半透明漸變（中央較亮）
- 字體：32pt 粗體
- 圓角：10px

### 按鈕
- 正常狀態：
  - 背景：深藍色漸變 (#0F2940 → #0D1F3C)
  - 文字：淺青白色 (#E8F4F8)
  - 邊框：霓虹青色 (#00FFFF) 3px
  - 圓角：10px
  - 字體：18pt 粗體
  
- Hover 狀態：
  - 背景：反轉漸變（更亮）
  - 邊框：亮青色 (#00D9FF)
  - 輕微發光效果
  
- Press 狀態：
  - 背景：純深藍色
  - 視覺反饋

## 互動流程

### 1. 啟動應用程式
```
應用程式圖標
    ↓ (點擊啟動)
主選單畫面
(四個按鈕，無選單欄)
```

### 2. 選擇本地遊玩
```
主選單
    ↓ (點擊「本地遊玩」)
遊戲畫面
    ├─ 顯示選單欄
    ├─ 顯示棋盤
    ├─ 自動開始遊戲
    └─ 白方先行
```

### 3. 進行遊戲
```
遊戲畫面
    ├─ 移動棋子
    ├─ 查看棋譜
    ├─ 調整時間
    └─ 可隨時返回主選單
```

### 4. 返回主選單
```
遊戲畫面
    ↓ (選單欄 → 遊戲 → 返回主選單)
確認對話框
    ├─ 是 → 主選單
    └─ 否 → 繼續遊戲
```

## 動畫效果（未實現，但可作為未來改進）

### 主選單動畫
1. 標題淡入 + 輕微縮放
2. 按鈕依次從下方滑入
3. Hover 時按鈕輕微放大 + 邊框發光

### 切換動畫
1. 主選單淡出
2. 遊戲畫面淡入
3. 總時長約 300-500ms

## 響應式設計

### 最小視窗尺寸
- 寬度：814px
- 高度：420px

### 按鈕大小
- 最小寬度：300px
- 內邊距：20px
- 字體：18pt

### 佈局調整
- 按鈕始終保持居中
- 上下留有適當間距
- 標題與按鈕之間有 30px 間距

## 無障礙功能

### 鍵盤支持
- Tab 鍵切換按鈕
- Enter/Space 啟動按鈕
- Esc 關閉對話框

### 視覺提示
- 明確的 Hover 狀態
- 清晰的 Focus 指示器
- 高對比度顏色
- 大字體易讀
