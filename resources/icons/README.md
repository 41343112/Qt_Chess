# Chess Piece Icon Sets

This directory contains three preset chess piece icon sets that players can choose from.

## Directory Structure

Each icon set directory (set1, set2, set3) should contain 12 PNG image files with transparent backgrounds:

### White Pieces:
- `white_king.png` - 白方國王 (♔)
- `white_queen.png` - 白方皇后 (♕)
- `white_rook.png` - 白方城堡 (♖)
- `white_bishop.png` - 白方主教 (♗)
- `white_knight.png` - 白方騎士 (♘)
- `white_pawn.png` - 白方兵 (♙)

### Black Pieces:
- `black_king.png` - 黑方國王 (♚)
- `black_queen.png` - 黑方皇后 (♛)
- `black_rook.png` - 黑方城堡 (♜)
- `black_bishop.png` - 黑方主教 (♝)
- `black_knight.png` - 黑方騎士 (♞)
- `black_pawn.png` - 黑方兵 (♟)

## Image Requirements

- **Format**: PNG (recommended for transparent background)
- **Size**: Recommended 128x128 pixels or larger (will be scaled automatically)
- **Background**: Transparent preferred
- **Style**: Each set should have a consistent visual style

## Icon Sets

### Set 1 (預設圖標集 1)
Classic traditional chess piece design

### Set 2 (預設圖標集 2)
Modern minimalist chess piece design

### Set 3 (預設圖標集 3)
Decorative artistic chess piece design

## Adding Your Images

Place your chess piece images in the appropriate set directory (set1, set2, or set3) following the naming convention above.
