# 吃子紀錄快速參考 (Captured Pieces Quick Reference)

## 快速查找 (Quick Lookup)

### 如何查看吃子紀錄 (How to View Captured Pieces)

```
步驟 (Steps):
1. 設定時間控制（非"無限制"）
2. 點擊"開始"按鈕
3. 走第一步棋（計時器啟動）
4. 吃子紀錄自動顯示 ✓
```

## 優勢計算範例 (Advantage Examples)

### 均勢 (Equal)
```
白方吃: ♟♟ (2分)
黑方吃: ♞ (3分)
差距: 3 - 2 = 1分

顯示:
白方: ♟♟
黑方: ♞ +1  ← 黑方領先1分
```

### 白方領先 (White Ahead)
```
白方吃: ♛ ♟ (10分)
黑方吃: ♜♜ (10分)
差距: 0分（但白方場上子力更多）

假設場上:
白方: ♕ ♖ ♖ ♗ ♗ ♘ ♘ ♙×6 = 40分
黑方: ♜ ♝ ♞ ♟×7 = 18分
差距: 40 - 18 = 22分

顯示:
白方: ♛ ♟ +22  ← 白方領先22分
黑方: ♜♜
```

## 重疊規則 (Overlapping Rules)

### 相同類型 = 重疊 (Same Type = Overlap)
```
輸入: 3個兵
顯示: ♟♟♟
     └─┴─ 重疊8px
```

### 不同類型 = 分開 (Different Types = Separate)
```
輸入: 兵、騎士、主教
顯示: ♟ ♞ ♝
     └─┘└─┘
     分開18px
```

### 混合顯示 (Mixed)
```
輸入: 兵、兵、騎士、騎士、騎士、主教
顯示: ♟♟ ♞♞♞ ♝
     └┘ └──┘ 
     重疊 重疊  分開
```

## 常見問題速查 (Quick FAQ)

### ❓ 為什麼看不到？
```
檢查清單:
□ 時間控制已設定（非無限制）
□ 已點擊"開始"按鈕
□ 已走至少一步棋
□ 遊戲尚未結束
```

### ❓ +N 怎麼算？
```
計算方式:
場上白方總子力 - 場上黑方總子力 = 差距

正數: 白方領先
負數: 黑方領先
零: 均勢（不顯示）
```

### ❓ 回放時會更新嗎？
```
✓ 會！
使用 ⏮ ◀ ▶ ⏭ 按鈕
吃子紀錄隨步數更新
```

## 實用技巧 (Useful Tips)

### 💡 技巧 1: 評估局面
```
快速判斷優劣勢:
看 +N 數字:
  +0 到 +2  = 均勢
  +3 到 +5  = 微弱優勢
  +6 到 +9  = 明顯優勢
  +10 以上  = 壓倒性優勢
```

### 💡 技巧 2: 決定交換
```
看到對方領先?
→ 尋找等價交換機會
→ 減少子力差距

領先對方?
→ 避免不利交換
→ 保持優勢
```

### 💡 技巧 3: 學習價值
```
記住口訣:
1 兵
3 小子（騎士、主教）
5 大車
9 皇后
```

### 💡 技巧 4: 回放分析
```
找出關鍵交換點:
1. 進入回放模式
2. 觀察 +N 變化
3. 找出轉折點
4. 學習改進
```
