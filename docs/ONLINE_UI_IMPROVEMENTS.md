## 操作流程對比

### 改進前（7步）
```
房主：
1. 點擊「線上」按鈕
2. 選擇「創建房間」
3. 點擊「確定」
4. 記下IP地址：192.168.1.100
5. 記下房號：1234
6. 用通訊軟體告訴朋友這兩個資訊
7. 等待加入

朋友：
1. 收到IP和房號
2. 點擊「線上」按鈕
3. 選擇「加入房間」
4. 手動輸入IP：192.168.1.100
5. 手動輸入房號：1234
6. 點擊確定
7. 等待連線
```

### 改進後（4步）
```
房主：
1. 點擊「線上」按鈕
2. 選擇「創建房間」→「開始」
3. 點擊「複製連線碼」
4. 用LINE傳給朋友

朋友：
1. 複製朋友傳來的連線碼
2. 點擊「線上」→ 選擇「加入」
3. 點擊「從剪貼簿貼上」
4. 點擊「開始」
```

**減少了 43% 的操作步驟！**

## 連線碼格式支援

系統支援多種連線碼格式，自動解析：

✅ `192.168.1.100:1234` （標準格式）
✅ `192.168.1.100 1234` （空格分隔）
✅ `連線碼：192.168.1.100:1234` （帶前綴）
✅ `Connection: 192.168.1.100:1234` （英文前綴）

系統會自動：
- 去除多餘空白
- 識別IP地址部分
- 提取4位數字房號
- 驗證格式正確性
- 填入對應欄位
