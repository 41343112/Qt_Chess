## 🚀 快速開始（3分鐘學會）

### 我要創建房間（給朋友）

1. 點擊 **🌐 線上** 按鈕
2. 選擇 **「我要創建房間」**
3. 點擊 **「開始」**
4. 點擊 **「📋 複製連線碼」**
5. 用LINE/WeChat傳給朋友

### 我要加入房間（朋友給我連線碼）

1. 複製朋友傳來的連線碼
2. 點擊 **🌐 線上** 按鈕
3. 選擇 **「我有連線碼」**
4. 點擊 **「📋 從剪貼簿貼上」**
5. 點擊 **「開始」**

## 📖 詳細功能說明

### 連線碼系統

**什麼是連線碼？**
連線碼就像是房間的鑰匙，包含了連線所需的所有資訊。

**格式範例：**
```
192.168.1.100:1234
```

**組成部分：**
- `192.168.1.100` = IP地址（房主的位置）
- `:` = 分隔符號
- `1234` = 房號（4位數字）

### 支援的格式

系統會自動識別以下格式：
```
✅ 192.168.1.100:1234
✅ 192.168.1.100 1234
✅ 連線碼：192.168.1.100:1234
✅ Connection: 192.168.1.100:1234
```

### 主要功能

| 功能 | 說明 |
|------|------|
| 📋 一鍵複製 | 創建房間後一鍵複製連線碼 |
| 📋 一鍵貼上 | 加入房間時一鍵貼上連線碼 |
| 🔍 自動解析 | 自動識別並填入IP和房號 |
| ✅ 格式驗證 | 自動檢查連線碼正確性 |
| 🎨 視覺引導 | 表情符號和顏色區分 |
| 📱 分享友善 | 可用任何通訊軟體分享 |

## ❓ 常見問題快速查詢

### 連線碼相關

**Q: 連線碼在哪裡？**
A: 創建房間後會顯示在大字體藍色框框中

**Q: 連線碼怎麼傳給朋友？**
A: 用任何通訊軟體都可以（LINE、WeChat、Messenger等）

**Q: 連線碼可以重複使用嗎？**
A: 每次創建房間都會生成新的連線碼

### 連線問題

**Q: 為什麼連不上？**
A: 最常見原因：
1. 連線碼輸入錯誤
2. 不在同一個WiFi
3. 防火牆阻擋

詳細解答請看：[簡易使用指南 - 常見問題](ONLINE_BEGINNER_GUIDE.md#常見問題)

### 使用疑問

**Q: 誰先走？**
A: 房主（創建房間的人）執白棋先走

**Q: 可以中途離開嗎？**
A: 可以，但遊戲會結束

**Q: 需要在同個地方嗎？**
A: 同個WiFi最簡單，不同地方需要進階設定

詳細說明請看：[簡易使用指南](ONLINE_BEGINNER_GUIDE.md)
