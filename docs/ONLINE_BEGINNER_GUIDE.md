# 線上對戰簡易使用指南

## 🎮 給不懂電腦的朋友：超簡單連線教學

這份指南用最簡單的方式教您如何跟朋友連線下棋，就像使用LINE一樣簡單！

**🌟 新版改進：現在使用中央伺服器，更簡單、更穩定！**
- ✅ 不需要知道IP地址
- ✅ 不需要設定路由器或防火牆
- ✅ 只要有房號就能連線

---

## 📱 第一步：決定誰創建房間

**兩個人中選一個當「房主」**
- 房主：會創建房間，拿到「房號」
- 朋友：會加入房間，需要輸入「房號」

💡 **提示**：誰當房主都可以，系統會自動處理連線

---

## 🏠 房主操作流程（創建房間的人）

### 1️⃣ 開啟遊戲，點擊「線上」按鈕

在遊戲主畫面右側控制面板，找到三個按鈕：
- 👥 雙人
- 🤖 電腦
- 🌐 **線上** ← 點這個！

### 2️⃣ 選擇「我要創建房間」

會出現一個對話框，有兩個選項：
- 📱 **我要創建房間（分享房號給朋友）** ← 選這個！
- 🔗 我有房號（加入朋友的房間）

選好後，點擊右下角的「✓ 開始」按鈕

### 3️⃣ 複製房號

系統會彈出一個視窗，裡面有一串房號，看起來像這樣：
```
1234
```

**這就是您的房號！**

點擊綠色的「📋 複製房號」按鈕，房號就會複製到您的剪貼簿

### 4️⃣ 傳送房號給朋友

打開您常用的通訊軟體（LINE、WeChat、Messenger等），把房號貼上並傳給朋友

```
例如傳送：
「來下棋！房號：1234」
```

### 5️⃣ 等待朋友加入

視窗會顯示「🔄 等待對手加入...」

朋友成功加入後，遊戲會自動開始！

您執白棋（白色棋子），先走第一步

---

## 🔗 朋友操作流程（加入房間的人）

### 1️⃣ 收到房主的房號

朋友會傳給您一串文字，看起來像：
```
1234
```

**先把這串文字複製起來**（長按文字 → 選擇「複製」）

### 2️⃣ 開啟遊戲，點擊「線上」按鈕

在遊戲主畫面右側控制面板，找到「🌐 線上」按鈕並點擊

### 3️⃣ 選擇「我有房號」

會出現對話框，選擇：
- 📱 我要創建房間（分享房號給朋友）
- 🔗 **我有房號（加入朋友的房間）** ← 選這個！

### 4️⃣ 貼上房號

選好後會看到兩種方式：

**方法一（推薦）：使用房號**
1. 找到「📋 方法一：使用房號（推薦）」區域
2. 點擊「📋 從剪貼簿貼上」按鈕
3. 系統會自動把您剛才複製的房號填進去

**方法二：手動輸入**
如果自動貼上沒成功，可以手動輸入：
1. 在「✏️ 方法二：手動輸入」區域
2. 把房號填進去
   - 例如房號 `1234`

### 5️⃣ 點擊「開始」

確認房號已填入後，點擊右下角「✓ 開始」按鈕

### 6️⃣ 等待連線

畫面會顯示「🔄 正在連接...」

連接成功後，遊戲自動開始！

您執黑棋（黑色棋子），等房主（白棋）走完第一步後輪到您

## ❓ 常見問題

### Q1：房號是什麼？
**A**：房號就像是房間的鑰匙，是由伺服器自動生成的唯一代碼
- 格式：4位數字（1000-9999）
- 例如：`1234`
- 由中央伺服器管理，無需設定網路

### Q2：我要去哪裡複製/貼上？
**A**：
- **複製**：在房間創建成功視窗，點擊「📋 複製房號」按鈕
- **貼上**：在加入房間畫面，點擊「📋 從剪貼簿貼上」按鈕

### Q3：房號複製後要怎麼傳給朋友？
**A**：用任何通訊軟體都可以！
- LINE
- WeChat
- Messenger
- WhatsApp
- 簡訊
- Email
只要能傳文字的都行！

### Q4：為什麼連不上？
**A**：最常見的原因：
1. **房號輸入錯誤**：仔細檢查每個字母和數字
2. **伺服器連線問題**：
   - 確認網路連線正常
   - 伺服器可能暫時維護中
   - ❌ 伺服器維護中 → 稍後再試
3. **房號已過期**：房號可能已失效，請房主重新創建

### Q5：我跟朋友不在同個地方能玩嗎？
**A**：**可以！這就是新版的優勢！**
- 🌟 使用中央伺服器，不管在哪都能連線
- ✅ 一個在台北，一個在高雄 → 沒問題！
- ✅ 一個在家，一個在學校 → 沒問題！
- ✅ 甚至跨國連線也可以！
- 📶 只要雙方都有網路就能玩

**不需要**：
- ❌ 不需要在同一個WiFi
- ❌ 不需要設定路由器
- ❌ 不需要知道IP地址
- ❌ 不需要設定防火牆

### Q6：誰先走？
**A**：
- 房主（創建房間的人）= 白棋 = 先走
- 朋友（加入房間的人）= 黑棋 = 後走

### Q7：可以中途離開嗎？
**A**：可以，但遊戲會結束。如果要繼續，需要重新創建房間

### Q8：為什麼我看到的棋盤方向不一樣？
**A**：這是正常的！
- 白棋玩家：白棋在下方
- 黑棋玩家：黑棋在下方（棋盤會自動翻轉）
這樣雙方都能從自己的角度看棋盤
