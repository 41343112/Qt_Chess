# 線上對戰簡易使用指南

## 🎮 給不懂電腦的朋友：超簡單連線教學

這份指南用最簡單的方式教您如何跟朋友連線下棋，就像使用LINE一樣簡單！

**🌟 新功能：中繼伺服器模式**
- 只需要4位數字房號就能連線
- 不用知道IP地址
- 可以跨不同網路（不同WiFi/不同地點都可以）
- 不需要設定路由器

---

## 📱 第一步：決定誰創建房間

**兩個人中選一個當「房主」**
- 房主：會創建房間，拿到「房號」
- 朋友：會加入房間，需要輸入「房號」

💡 **建議**：誰當房主都可以，使用中繼伺服器模式無所謂

---

## 🏠 房主操作流程（創建房間的人）

### 1️⃣ 開啟遊戲，點擊「線上」按鈕

在遊戲主畫面右側控制面板，找到三個按鈕：
- 👥 雙人
- 🤖 電腦
- 🌐 **線上** ← 點這個！

### 2️⃣ 確認「使用中繼伺服器」已勾選（推薦）

會出現一個對話框，確保：
- ✅ **「使用中繼伺服器」** 已勾選（預設就是勾選的）
  - 這樣才能跨網域連線，只需要房號

### 3️⃣ 選擇「我要創建房間」

有兩個選項：
- 📱 **我要創建房間（取得房號給朋友）** ← 選這個！
- 🔗 我有房號（加入朋友的房間）

選好後，點擊右下角的「✓ 開始」按鈕

### 4️⃣ 取得房號

系統會彈出一個視窗，裡面顯示一個4位數字的房號，看起來像這樣：
```
1234
```

**這就是您的房號！**

點擊綠色的「📋 複製房號」按鈕，房號就會複製到您的剪貼簿

### 5️⃣ 傳送房號給朋友

打開您常用的通訊軟體（LINE、WeChat、Messenger等），把房號貼上並傳給朋友

```
例如傳送：
「來下棋！房號：1234」
```

### 6️⃣ 等待朋友加入

視窗會顯示「🔄 等待對手加入...」

朋友成功加入後，遊戲會自動開始！

您執白棋（白色棋子），先走第一步

---

## 🔗 朋友操作流程（加入房間的人）

### 1️⃣ 收到房主的房號

朋友會傳給您一個4位數字，看起來像：
```
1234
```

**記住這個房號**（或先複製起來）

### 2️⃣ 開啟遊戲，點擊「線上」按鈕

在遊戲主畫面右側控制面板，找到「🌐 線上」按鈕並點擊

### 3️⃣ 確認「使用中繼伺服器」已勾選

會出現對話框，確保：
- ✅ **「使用中繼伺服器」** 已勾選（預設就是勾選的）

### 4️⃣ 選擇「我有房號」

選擇：
- 📱 我要創建房間（取得房號給朋友）
- 🔗 **我有房號（加入朋友的房間）** ← 選這個！

### 5️⃣ 輸入房號

在「輸入房號」欄位中，輸入朋友給您的4位數字房號：
```
例如：1234
```

### 6️⃣ 點擊「開始」

確認房號已填入後，點擊右下角「✓ 開始」按鈕

### 7️⃣ 等待連線

畫面會顯示「🔄 正在連接...」

連接成功後，遊戲自動開始！

您執黑棋（黑色棋子），等房主（白棋）走完第一步後輪到您

## ❓ 常見問題

### Q1：房號是什麼？
**A**：房號就像是房間的鑰匙，是一個4位數字
- 格式：4位數字（例如：`1234`）
- 使用中繼伺服器模式時，只需要房號就能連線
- 不需要知道IP地址

### Q2：我要去哪裡複製/輸入房號？
**A**：
- **房主**：在房間創建成功視窗，點擊「📋 複製房號」按鈕
- **加入者**：在加入房間畫面，直接輸入4位數字房號

### Q3：房號複製後要怎麼傳給朋友？
**A**：用任何通訊軟體都可以！
- LINE
- WeChat
- Messenger
- WhatsApp
- 簡訊
- Email
只要能傳文字的都行！

### Q4：為什麼連不上？
**A**：最常見的原因：
1. **房號輸入錯誤**：確認是正確的4位數字
2. **中繼伺服器未運行**：需要中繼伺服器正在運行（參見[中繼伺服器設定指南](RELAY_SERVER_SETUP.md)）
3. **網路連線問題**：檢查網路連線是否正常

💡 **使用中繼伺服器模式的優點**：
- ✅ 不同地方（不同WiFi）都可以連
- ✅ 不需要設定路由器
- ✅ 不需要知道IP地址
- ✅ 更簡單方便

### Q5：我跟朋友不在同個地方能玩嗎？
**A**：可以！使用中繼伺服器模式就可以：
- ✅ 一個在家，一個在公司 → 可以連
- ✅ 一個在台北，一個在高雄 → 可以連
- ✅ 不同WiFi → 可以連

**前提**：需要有中繼伺服器正在運行（可以自己架設或使用公開的）

### Q6：什麼是中繼伺服器？
**A**：中繼伺服器就像是一個「中間人」，幫助兩個玩家連線
- 玩家不需要知道對方的IP地址
- 只需要房號就能找到對方
- 類似LINE、WeChat的伺服器功能

想了解更多，請參閱 [中繼伺服器設定指南](RELAY_SERVER_SETUP.md)

### Q7：誰先走？
**A**：
- 房主（創建房間的人）= 白棋 = 先走
- 朋友（加入房間的人）= 黑棋 = 後走

### Q8：可以中途離開嗎？
**A**：可以，但遊戲會結束。如果要繼續，需要重新創建房間

### Q9：為什麼我看到的棋盤方向不一樣？
**A**：這是正常的！
- 白棋玩家：白棋在下方
- 黑棋玩家：黑棋在下方（棋盤會自動翻轉）
這樣雙方都能從自己的角度看棋盤

## 🎯 快速開始檢查清單

- [ ] 選擇「我要創建房間」

- [ ] 等待朋友加入

### 朋友（加入房間）
- [ ] 收到並複製房主的連線碼
- [ ] 點擊「🌐 線上」按鈕
- [ ] 選擇「我有連線碼」
- [ ] 點擊「從剪貼簿貼上」
- [ ] 點擊「開始」
- [ ] 開始下棋！（您是黑棋，後走）

## 💡 小技巧

1. **第一次使用**：建議兩個人坐在一起試試看，確保知道怎麼操作
2. **連線碼保管**：可以把連線碼存在記事本，下次就不用重新創建
3. **同個WiFi**：在同一個WiFi下連線最穩定，不會斷線
4. **測試連線**：可以先在家裡的兩台設備測試看看
