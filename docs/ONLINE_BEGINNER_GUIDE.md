# 線上對戰簡易使用指南

## 🎮 給不懂電腦的朋友：超簡單連線教學

這份指南用最簡單的方式教您如何跟朋友連線下棋，就像使用LINE一樣簡單！

---

## 📱 第一步：決定誰創建房間

**兩個人中選一個當「房主」**
- 房主：會創建房間，拿到「連線碼」
- 朋友：會加入房間，需要輸入「連線碼」

💡 **建議**：選網路比較穩定的人當房主

---

## 🏠 房主操作流程（創建房間的人）

### 1️⃣ 開啟遊戲，點擊「線上」按鈕

在遊戲主畫面右側控制面板，找到三個按鈕：
- 👥 雙人
- 🤖 電腦
- 🌐 **線上** ← 點這個！

### 2️⃣ 選擇「我要創建房間」

會出現一個對話框，有兩個選項：
- 📱 **我要創建房間（給朋友連線碼）** ← 選這個！
- 🔗 我有連線碼（加入朋友的房間）

選好後，點擊右下角的「✓ 開始」按鈕

### 3️⃣ 複製連線碼

系統會彈出一個視窗，裡面有一串連線碼，看起來像這樣：
```
192.168.1.100:1234
```

**這就是您的連線碼！**

點擊綠色的「📋 複製連線碼」按鈕，連線碼就會複製到您的剪貼簿

### 4️⃣ 傳送連線碼給朋友

打開您常用的通訊軟體（LINE、WeChat、Messenger等），把連線碼貼上並傳給朋友

```
例如傳送：
「來下棋！連線碼：192.168.1.100:1234」
```

### 5️⃣ 等待朋友加入

視窗會顯示「🔄 等待對手加入...」

朋友成功加入後，遊戲會自動開始！

您執白棋（白色棋子），先走第一步

---

## 🔗 朋友操作流程（加入房間的人）

### 1️⃣ 收到房主的連線碼

朋友會傳給您一串文字，看起來像：
```
192.168.1.100:1234
```

**先把這串文字複製起來**（長按文字 → 選擇「複製」）

### 2️⃣ 開啟遊戲，點擊「線上」按鈕

在遊戲主畫面右側控制面板，找到「🌐 線上」按鈕並點擊

### 3️⃣ 選擇「我有連線碼」

會出現對話框，選擇：
- 📱 我要創建房間（給朋友連線碼）
- 🔗 **我有連線碼（加入朋友的房間）** ← 選這個！

### 4️⃣ 貼上連線碼

選好後會看到兩種方式：

**方法一（推薦）：使用連線碼**
1. 找到「📋 方法一：使用連線碼（推薦）」區域
2. 點擊「📋 從剪貼簿貼上」按鈕
3. 系統會自動把您剛才複製的連線碼填進去

**方法二：手動輸入**
如果自動貼上沒成功，可以手動輸入：
1. 在「✏️ 方法二：手動輸入」區域
2. 把IP地址和房號分別填進去
   - 例如連線碼 `192.168.1.100:1234`
   - IP地址填：`192.168.1.100`
   - 房號填：`1234`

### 5️⃣ 點擊「開始」

確認連線碼已填入後，點擊右下角「✓ 開始」按鈕

### 6️⃣ 等待連線

畫面會顯示「🔄 正在連接...」

連接成功後，遊戲自動開始！

您執黑棋（黑色棋子），等房主（白棋）走完第一步後輪到您

## ❓ 常見問題

### Q1：連線碼是什麼？
**A**：連線碼就像是房間的鑰匙，包含了房主的「地址」和「房號」
- 格式：`IP地址:房號`
- 例如：`192.168.1.100:1234`

### Q2：我要去哪裡複製/貼上？
**A**：
- **複製**：在房間創建成功視窗，點擊「📋 複製連線碼」按鈕
- **貼上**：在加入房間畫面，點擊「📋 從剪貼簿貼上」按鈕

### Q3：連線碼複製後要怎麼傳給朋友？
**A**：用任何通訊軟體都可以！
- LINE
- WeChat
- Messenger
- WhatsApp
- 簡訊
- Email
只要能傳文字的都行！

### Q4：為什麼連不上？
**A**：最常見的原因：
1. **連線碼輸入錯誤**：仔細檢查每個數字和符號
2. **不在同一個網路**：
   - ✅ 同一個WiFi → 一定可以連
   - ✅ 同一個家裡/辦公室 → 通常可以連
   - ❌ 不同地方（一個在家，一個在公司）→ 需要進階設定
3. **防火牆阻擋**：關閉防火牆或允許程式通過

### Q5：我跟朋友不在同個地方能玩嗎？
**A**：可以，但需要一些設定：
- 如果您們在不同WiFi（例如一個在家、一個在學校），需要：
  1. 房主需要知道自己的「公網IP」
  2. 房主需要在路由器設定「端口轉發」
  
**建議**：如果是第一次使用，先在同個WiFi下試試看！

### Q6：誰先走？
**A**：
- 房主（創建房間的人）= 白棋 = 先走
- 朋友（加入房間的人）= 黑棋 = 後走

### Q7：可以中途離開嗎？
**A**：可以，但遊戲會結束。如果要繼續，需要重新創建房間

### Q8：為什麼我看到的棋盤方向不一樣？
**A**：這是正常的！
- 白棋玩家：白棋在下方
- 黑棋玩家：黑棋在下方（棋盤會自動翻轉）
這樣雙方都能從自己的角度看棋盤

## 💡 小技巧

1. **第一次使用**：建議兩個人坐在一起試試看，確保知道怎麼操作
2. **連線碼保管**：可以把連線碼存在記事本，下次就不用重新創建
3. **同個WiFi**：在同一個WiFi下連線最穩定，不會斷線
4. **測試連線**：可以先在家裡的兩台設備測試看看
