## 使用流程圖 (Usage Flow)

```
開始
  │
  ├─→ 選擇時間限制 (總時間下拉選單)
  │   ├─ 無限制 → 計時器顯示 --:--
  │   └─ 有時間限制 → 設定初始時間
  │
  ├─→ 設定增量 (每著加秒滑桿)
  │   └─ 0-60 秒範圍
  │
  ├─→ 點擊「新遊戲」
  │   └─ 重置計時器
  │
  ├─→ 走第一步棋
  │   └─ 計時器開始倒數 (活躍玩家變綠色)
  │
  ├─→ 繼續走棋
  │   ├─ 每走一步後
  │   │  ├─ 增加增量時間到剛走棋的玩家
  │   │  └─ 切換到對手計時
  │   │
  │   └─ 檢查時間
  │      ├─ 時間足夠 → 繼續遊戲
  │      └─ 時間歸零 → 遊戲結束
  │
  └─→ 遊戲結束
      └─ 可以點擊「新遊戲」重新開始
```

## 鍵盤快捷鍵建議 (Suggested Keyboard Shortcuts)

雖然當前實現沒有包含鍵盤快捷鍵，但可以考慮在未來版本中添加：

- `Ctrl+N` - 新遊戲
- `Ctrl+P` - 暫停/恢復計時器
- `Ctrl+T` - 開啟時間控制設定
- `Space` - 快速切換時間限制選項
