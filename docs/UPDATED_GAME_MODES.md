## 更新內容

### 五種新遊戲模式

1. **🌫️ 霧戰** (Fog of War) - 默認勾選
   - 玩家只能看到部分棋盤
   
2. **🌍 地吸引力** (Gravity)
   - 棋子受重力影響移動
   
3. **🔮 傳送陣** (Teleportation)
   - 棋子可以傳送到特定位置
   
4. **🎲 骰子** (Dice)
   - 使用骰子決定移動規則
   
5. **💣 踩地雷** (Landmine)
   - 棋盤上有隱藏地雷

### 樣式更新

- **文字顏色**: 所有複選框文字改為白色 (`color: white;`)
- **圖標**: 使用新的表情符號圖標
- **尺寸**: 保持 400x350 像素

## 數據對應

在程式中，這些模式會以以下名稱儲存：

```cpp
gameModes["霧戰"] = true/false
gameModes["地吸引力"] = true/false
gameModes["傳送陣"] = true/false
gameModes["骰子"] = true/false
gameModes["踩地雷"] = true/false
```
